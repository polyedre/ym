QUICK COMMANDS
--------------

Search for passwords:
  $ ym grep tests/data/app-config.yaml '.*\.password'

Find API ports:
  $ ym grep tests/data/openstack-config.yaml '.*api\.port'

Find feature toggles:
  $ ym grep tests/data/feature-flags.yaml '.*\.enabled'

Update replica count:
  $ ym set tests/data/helm-values.yaml replicaCount=5

Update multiple values:
  $ ym set tests/data/config-prod.yaml database.primary.port=5433 app.log_level=warn

Remove a key:
  $ ym unset tests/data/feature-flags.yaml features.experimental

COMMON PATTERNS
---------------

Find all passwords:
  ym grep <file> '.*\.password'

Find database hosts:
  ym grep <file> '.*\.db\.host'

Find API configurations:
  ym grep <file> '.*\.api\..*'

Find enabled features:
  ym grep <file> '.*\.enabled'

Find with specific value:
  ym grep <file> '.*=true'

UPDATE EXAMPLES
---------------

Single update:
  ym set file.yaml key.path=new_value

Multiple updates:
  ym set file.yaml key1=val1 key2=val2 key3=val3

Create new nested path:
  ym set file.yaml new.section.key=value
